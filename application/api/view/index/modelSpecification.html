一、原则

1.颜饭致力于为用户提供积极、健康、绿色、和谐的网络环境。为维护颜饭运营秩序，保障用户合法权益，为用户提供良好体验，颜饭根据中国现行有效的法律法规及《颜饭用户协议》，制定《颜饭用户行为规范》（以下简称“规范”）。

2.用户应当遵守国家相关法律法规以及颜饭平台的协议规则和行为规范，违反相关法律法规和颜饭规则的，颜饭会做出必要的处理。

二、注册规范

1.用户在注册成为颜饭正式用户中，应当填写真实有效的手机号，根据颜饭的要求完成验证。

2.用户对其注册的颜饭帐号享有使用权，用户不得将注册的颜饭帐号赠与、借用、租用、转让或售卖。

3.用户帐号名称、头像等资料不得涉及色情、赌博、吸毒、暴力等法律法规禁止的内容。

4.不得在帐号名称、头像中宣传任何广告相关信息。包括但不限于在发布的图片中包含自己或他人的微博帐号、微信帐号、微信公众号、淘宝店铺、微商、整容整形、贷款、信用卡、化妆品等广告相关信息。

5.用户帐号名称、头像等资料不得涉及侵害他人名誉权、肖像权、知识产权、商业秘密等合法权利。

三、使用规范

1.用户不得发布色情及色情擦边内容。包括但不限于：

（1）散布淫秽、色情内容，包括但不限于招嫖、约炮等内容；

（2）发布带有色情暗示、挑逗性的文字、图片、外部网站链接和二维码等内容。

2.用户不得在颜饭上发布任何广告相关信息。包括但不限于：

（1）发布带有水印的图片；

（2）发布含有自己或他人的微博帐号、微信帐号、微信公众号、淘宝店铺、微商等宣传信息的图片

（3）发布含有整容整形、贷款、信用卡、化妆品等广告相关信息的图片。

3.用户不得盗用他人的资料。包括但不限于盗用他人享有著作权的图片、企业的名称、商标等。

4.用户不得侵犯他人的肖像权、名誉权、著作权、商标权等合法权益。包括但不限于未经他人允许或授权，非法使用、传播、恶搞他人的肖像、照片等。

5.用户行为不得违法国家现行法律法规。包括但不限于：

（1）反对宪法所确定的基本原则、危害国家安全、颠覆国家政权、煽动民族仇恨、宣扬邪教和封建迷信、非法集会等；

（2）发布买卖假币、枪支弹药以及管制刀具、买卖违禁药品、假证等信息；

（3）发布买卖毒品、相约吸毒、聚众吸毒等涉及毒品的信息；

（4）发布含有聚众赌博、传授赌博的内容；

（5）发布引诱他人从事恐怖、暴力等信息或者发布的内容中含有恐怖、暴力信息；

6.用户不得发布血腥、暴力、恐怖、恶心等会令人感到不适的内容。包括但不限于：

（1）散布动物被杀、致残以及枪击、刺伤、拷打等虐待动物的真实画面；

（2）发布含有描绘暴力或虐待儿童等内容；

（3）发布替人复仇、收账、炫耀黑社会身份等具有黑社会性质的信息.

7.用户不得发布钓鱼网站链接、含木马病毒的网站链接和二维码等可能会对其他用户造成信息安全影响的内容。

8.用户不得使用外挂或者其他恶意行为，损害颜饭或颜饭用户的合法权益。该行为包括但不限于：

（1）未经颜饭书面许可使用插件、外挂或其他第三方工具、服务接入颜饭以及相关系统

（2）恶意攻击颜饭及相关系统；

（3）恶意收集颜饭系统或者颜饭其他用户的个人信息；

（4）编造、散布虚假信息，恶意中伤颜饭商业信誉与名誉；

（5）其他恶意损害颜饭合法权利、经济利益的行为。

四、颜饭的处理措施

1.针对用户在注册颜饭帐号过程中的做出的违反本规范的行为，颜饭将不予准许用户注册颜饭帐号；

2.针对用户在使用颜饭中做出的违反本规范的行为，颜饭将禁止用户使用帐号的部分功能；

3.用户的为规行为已经或涉嫌侵犯颜饭、颜饭其他用户、国家、集体合法利益的，颜饭将冻结用户帐号、封禁帐号或注销帐号。

4.用户对颜饭做出的处理措施有疑问或异议的，可以立即与颜饭联系，颜饭将根据有关规则做出解释或处理。
